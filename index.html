Optimization recommendations

Never use pictures, when you can cope without them.

CSS Image Sprites.

In fixed blocks with fixed size you need to choose the pictures strictly corresponding to block size.


For example, loading 300px * 300px picture to a
200px * 200px block, we increase the image file size in comparison to the required one by 125%.

Don't forget to specify the description of the image. Use the alt attribute for this.

The key point in optimization is a size compression



SVG contains in its code some metadata, for example, information about layers, comments and XML namespaces, which is no use to display in the browser.
It's better to remove all unnecessary info.

In this case we may use the tool for optimizing SVG files "svgo".

github.com/svg/svgo

SVG can be compressed to GZIP
